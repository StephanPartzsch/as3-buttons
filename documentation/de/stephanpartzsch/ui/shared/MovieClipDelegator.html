Classpublic class MovieClipDelegator
InheritanceMovieClipDelegator flash.display.MovieClip
Subclasses FrameButton

Creates an interface so that any objects that inherits from the MovieClipDelegator is able to delegate every call to any method and any property of type MovieClip to a given MovieClip instance.

addMovieClip()method 
public function addMovieClip(movieClipInstance:MovieClip, addToDisplayList:Boolean = false):void

Adds a instance of type MovieClip to this MovieClipDelegator. Optionally this instance can be added as a child to the display list of the super class.

Parameters

movieClipInstance:MovieClip — The instance of type MovieClip to which every call is delegated.
 
addToDisplayList:Boolean (default = false) — If movieClipInstance should be added to the display list of the super class. Default is false.


Throws
ArgumentError — If the given movieClipInstance is null.
