InheritanceMissingLabelError Error

The MissingLabelError class represents an error that occurs if a type of FrameButton is instantiated with a MovieClip instance, that does not contain the required labels. It creates an error message which looks like this: One of the required frame labels (...) is missing for the button instance (...) of type FrameButton.
