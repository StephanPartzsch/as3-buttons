Interfacepublic interface IFrameButton
Implementors FrameButton, ToggleFrameButton

The IFrameButton interface defines methods for the enabling, disabling, selecting, deselecting and disposing a FrameButton as well as for returning the MovieClip instance which is used within a FrameButton.
